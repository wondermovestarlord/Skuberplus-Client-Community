# Changelog

All notable changes to this project will be documented in this file.

## [0.6.5] - 2026-04-14

### Added

- Initial open-source release
- AI-powered Kubernetes diagnostics with LangChain integration
- Real-time cluster monitoring and metrics dashboard
- Multi-cluster management with unified sidebar
- Kubernetes resource CRUD operations (Pods, Deployments, Services, etc.)
- Built-in terminal with kubectl integration
- Helm chart management
- Custom Resource Definition (CRD) support
- Dark/Light theme support
- Multi-language UI support

### Based On

- Forked from [Open Lens](https://github.com/lensapp/lens) with significant enhancements
